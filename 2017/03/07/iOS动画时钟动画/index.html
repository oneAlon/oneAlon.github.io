iOS动画:时钟动画

发表于 2017-03-07

准备学习下iOS的动画,写几篇文章梳理下知识

知识补充

在开始之前先来了解一些知识CALayer CADisplayLink position anchorPoint transform 隐式动画

  • CALayer

    每个UIView都有一个layer属性, UIView本身不具备显示的功能,UIView能够显示出来主要是它的layer属性,当UIView需要显示出来的时候就会调用(void)drawRect:(CGRect)rect 方法,在drawRect方法中将需要显示出来的内容绘制到自己的layer图层中显示出来.

    通过操作CALayer可以很方便的调整UIView的外观阴影 圆角 边框大小和颜色 等,还可以给UIView添加一些非常酷炫的动画.在CALayer中不能使用UIImage和UIColor,只能使用CGImageRef和CGColorRef.

    CALayer的一些属性(具体属性可在CALayer头文件查看):

    CALayer和UIView如何选择:

    • CALayer和UIView能够实现相同的显示效果,但是CALayer不能处理事件,而UIView可以

    • CALayer的性能会比UIView性能高,因为CALayer少了处理事件的功能

      ​

  • CADisplayLink

    CADisplayLink是QuartzCore.framework 中的一个类,能够让我们和屏幕刷新频率同步的将内容绘制到屏幕上,iOS的屏幕刷新频率为60次/s,所以CADisplayLink的selector方法没60s调用一次,也可以通过设置CADisplayLink的属性frameInterval修改.

    CADisplayLink和NSTimer比较:CADisplayLink是一个计时器，可以使绘图代码与视图的刷新频率保持同步，而NSTimer无法确保计时器实际被触发的准确时间.

  • position 和 anchorPoint

    position决定了layer在父控件中的位置

    anchorPoint决定了相对于layer的哪个位置定位,是layer的一个属性

  • transform

    从layer的锚点anchorpoint到给定的点连一根线,layer围绕这根线做旋转(此时的坐标轴为xyz三个方向)

    上句代码意思,从minLayer的锚点到(0,0,1)连根线,围绕这根线旋转45度,也就是围绕Z轴旋转45度

  • 隐式动画

    每一个UIView内部都默认关联着一个CALayer，我们可用称这个Layer为Root Layer(根层),我们手动创建的CALayer对象(非RootLayer)都存在着隐式动画

    隐式动画:对非RootLayer的一些属性进行修改时,会自动产生一些动画效果,可以通过事务CATransaction关闭默认的隐式动画

    bounds：用于设置CALayer的宽度和高度。修改这个属性会产生缩放动画

    backgroundColor：用于设置CALayer的背景色。修改这个属性会产生背景色的渐变动画

    position：用于设置CALayer的位置。修改这个属性会产生平移动画
